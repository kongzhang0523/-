# 梦幻资产管家

基于《梦幻西游电脑版》的游戏资产与收益管理专业工具。

## 🚀 功能特色

### 核心功能
- **游戏会话管理** - 精准记录游戏时间与点卡成本
- **智能流水记录** - 支持事后补录与关联会话
- **多开成本核算** - 自动计算多角色点卡消耗
- **数据可视化看板** - 专业级收益趋势与效率分析
- **资产清单管理** - 全面掌握游戏资产分布

### 用户体验
- **混合模式** - 先体验后注册，数据无缝迁移
- **响应式设计** - 支持桌面端与移动端
- **本地缓存** - 未登录状态下数据安全保存

## 🛠 技术栈

### 前端
- React 18 + Vite
- Ant Design 5.x
- ECharts 数据可视化
- Context API 状态管理

### 后端  
- Node.js + Express
- MongoDB + Mongoose
- JWT 身份认证
- RESTful API 设计

## 📦 快速开始

### 环境要求
- Node.js 16+
- MongoDB 5+

### 本地开发
1. 前端启动：
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

2. 后端启动：
\`\`\`bash
cd backend  
npm install
npm run dev
\`\`\`

## 📱 使用指南

1. **首次使用**：直接开始记录，无需注册
2. **记录游戏**：点击"开始游戏"，设置多开数量
3. **补录收益**：游戏结束后在结算界面批量补录
4. **数据迁移**：体验满意后注册账号，数据自动同步到云端
